# 不改变原数组

函数 说明 示例
Array.isArray() 判断一个值是否为数组 Array.isArray(arr); // true
join() 返回数组成员拼接后的字符串，默认逗号分隔 arr.join("-"); // "1-2-3"
concat() 返回多个数组合并后的新数组 arr.concat([4, 5], 6); // [1, 2, 3, 4, 5, 6]
slice() 返回切片后的新数组浅拷贝 arr.slice(start, end);
forEach() 参数为回调函数，会遍历数组所有的项，回调函数接受三个参数，分别为value，index，self；forEach没有返回值 arr.forEach(element => console.log(element));
map() 同forEach，同时回调函数返回数据，组成新数组由map返回 const map1 = arr.map(x => x * 2);
every() 同forEach，同时回调函数返回布尔值，全部为true，由every返回true arr.every((element, index, arr)=> {return true;});
some() 同forEach，同时回调函数返回布尔值，只要有一个为true，由some返回true，有true就跳出循环 list.some(item => item > 0) console.log(res) // true
reduce() 归并，同forEach，迭代数组的所有项，并构建一个最终值，由reduce返回 arr.reduce((total, element, index, arr)=> {return total + =element;},0);
reduceRight() 反向归并，同forEach，迭代数组的所有项，并构建一个最终值，由reduceRight返回
filter() 同forEach，同时回调函数返回布尔值，为true的数据组成新数组由filter返回
indexOf() 开头查找元素返回元素索引，没有返回-1 arr.indexOf(1); // 0

# 改变原数组

函数 说明 示例
push() 数组尾部添加元素，并返回数组长度 arr.push(1, 2); // 5
pop() 删除数组的最后一个元素，并返回该元素 arr.pop(); // 3
unshift() 数组头部添加元素，并返回数组长度 arr.unshift('a'); //4
shift() 删除数组的第一个元素，并返回该元素 arr.shift(); // 1
reverse() 返回颠倒顺序后的数组 arr.reverse(); // [3, 2, 1]
splice() 删除数组成员，并添加的数组成员，返回被删除的元素 arr.splice(start, deleteCount, ...itmes)
sort() 数组排序 arr.sort(); //[1, 2, 3]
slice(begin，end) 返回新数组， begin 和 end 决定的原数组的浅拷贝（包括 begin，不包括end），end没有就是往后无限取 arr.slice(2)//[3]

# ES6 数组方法

函数 说明 示例
Array.from() 类数组转换成数组 Array.from('123'); // ["1", "2", 3"]
Array.of() 将一组值，转换为数组 Array.of(1, 2, 3); // [1, 2, 3]
find() 返回满足提供的测试函数的第一个元素值。否则返回 [undefined]，不会遍历所有数组 arr.find((item)=>item===1)
findIndex() 查找元素位置，没有返回-1 arr.findIndex((item)=>item===1)
includes() 是否包含给定的值 arr.includes(1); // true
...arr 扩展运算符可以将伪数组变为数组

    return !arr2.find(itemArr2 => {

      param.forEach(item => {
